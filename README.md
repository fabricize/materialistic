# Materialistic

Clarify materials by MPN, SKU and vague material name.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'materialistic'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install materialistic

## Usage

```ruby
materials = Materialistic.find "LCD-10862"
p materials
```
